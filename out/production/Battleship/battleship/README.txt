Здравствуйте!
Ознакомьтесь, пожалуйста, перед использованием программы с README.txt


Ввод с командной строки:
Сначала вводятся через пробел два числа (размеры поля), затем опять через пробел 
количество кораблей (ВНИМАНИЕ: сначала вводятся однопалубные, потом двухпалубные и т.д. до пятипалубных)
Пример:
18 17 1 2 3 4 5
Расшифровка примера:
Создается поле размера 18(высота) на 17(ширина)
Далее вводятся корабли, то есть один "Submarine(1 cell)", два "Destroyer(2 cell)", три "Cruiser (3 cells)",
четыре "Battleship(4 cells)", пять "Carrier(5 cells)".


Ввод с консоли:
Программа будет выводить сообщения о необходимых действиях.


Расшифровка знаков:
"-" - не выстрелил (not-fired)
"^" - выстрелил-промахнулся(fired-miss)
"@" - выстрелил-попал(fired-hit)
"x" - потонул(sunk)
